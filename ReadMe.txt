----------------------------------------------------------------------------------

				Cities Box with Pacriman

----------------------------------------------------------------------------------

1.はじめに
　Cities Box with Pacrimanをダウンロードしていただきありがとうございます。
　本書では、Cities Box with Pacrimanの操作法や扱い方などについて簡単に説明いたします。

2.ソースコードについて
　ソースコードは是非改造にご利用ください。
　また、ソースコード上で何かアドバイスなどがありましたら
　コンタクトフォームなどで教えてくださるとありがたいです。

3.使用方法
　Cities_Box.exeを実行するとご利用いただけます。
　または、Cities Box.hspをコンパイルしてお使いください。

4.ルール
　パックリマンを十字キー動かして、道路上に散らばる餌をできるだけ多く食べさせましょう。
　ただし、おばけにぶつかるとゲームオーバーです。
　そのかわり、大きい餌を食べると無敵になり、おばけを食べることができます。
　大きい餌を食べてから20マス動くと、無敵モードが解除されます。

5.操作方法
　5.1 メニュー画面
　　上下キーでカーソルを移動し、Enterキーで決定します。
　
　5.2 ゲーム中
　　十字キーでパックリマンを動かしてください。
　　※パックリマンが移動できるのは道路上だけです。

6.補足
　いちおう通常のCities Boxの機能も利用できます。
　また、data/config.datのPacriman_Mode = 1;をPacriman_Mode = 0;にすると通常のCities Boxとして遊べます。たぶん…
